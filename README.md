# PRODIGY_DS_01
Data Science Internship Task 01 - Data Visualization using Python

## Task
Create a bar chart or histogram to visualize the distribution of a categorical or continuous variable.

## Dataset
Titanic Dataset (train_and_test2.csv)
 you can download it from here:https://www.kaggle.com/datasets/yasserh/titanic-dataset
## Technologies Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

## Visualizations
### 1. Histogram of Passenger Ages
A histogram with a Kernel Density Estimate (KDE) curve was created to visualize the distribution of passenger ages.

### 2. Bar Chart of Passenger Gender
A bar chart was created to compare the number of male and female passengers.

## Observations
- Most passengers were between 20 and 30 years old.
- The age distribution is positively skewed.
- The dataset contains more male passengers than female passengers.
